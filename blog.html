Blog Questions and Answers (Q&A)

1. Discuss the scope of var, let, and const.

Ans: In JavaScript there are 3 types of a variable declaration. 'Let' and 'Const' are 2 ES6 concepts. Previously 'Var' was used. The scope of Var is global, when it's declared outside a function, otherwise when var is declared within a function it's functional scoped. On the other hand, Let and Const both are block scoped. By declaring let, the variable value can changeble, whereas, declaration const means the variable is constant or is not changeable. Now a days, var is not but let and const are highly used.

2. Tell us the use cases of null and undefined.

Ans: 'Null' means an empty value. Otherwise, 'Undefined' means the variable has been declared, but its value has not been assigned. Actually, undefined is the default value assigned by JavaScript to uninitialized variables. Otherwise, a null value in JavaScript is used to indicate an object value is absent actually.

3. What do you mean by REST API ?

Ans: REST API means 'Representational State Transfer' Application Programming Interface (API). It means every url represents some objects. REST API transfers client to server and server to client some state or condition via representation.
